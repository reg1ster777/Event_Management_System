欢迎使用活动管理系统

本系统面向高校场景，提供从 活动创建 → 在线报名 → 现场签到 → 结果统计 的一体化管理流程。

当前已实现的核心能力包括： 管理员登录与管理、活动增删改查、学生在线报名、签到记录查询等。
